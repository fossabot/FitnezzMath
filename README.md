FitnezzMath is a repository of Fitnezz.io calculus and classifications.


Calculus: 
* Body Mass Index (BMI);
* Waist to Hip Ratio (WHR);
* Fat Percentage;
* Residual Weight;
* Fat Weight;
* Lean Body Mass (LBM);
* BoneWeight
